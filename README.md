# Sfiral Engine: ДНК Искусственного Интеллекта

Экспериментальная архитектура ИИ, основанная на Антисимметрии и S-Логике.
На основе исследований Олега Басаргина.

---

## О Проекте

Sfiral Engine — это первая попытка создать Топологический ИИ, основанный не на статистике, а на законах физики смыслов. В основе проекта лежит теория Сфирали (Sfiral) — особой геометрической структуры, имеющей S-образную форму и свойство антисимметрии.

## Почему это "ДНК ИИ"?

Мы называем эту архитектуру "Цифровой ДНК", потому что она копирует фундаментальный принцип жизни — двойную спираль.

В Sfiral Engine любая единица информации (Бингл) существует только как напряжение между двумя встречными потоками:
1. Тезис (V+) — устремленность в будущее (созидание).
2. Антитезис (V-) — память прошлого (отражение).

Бингл (Bingle) — это ячейка памяти, которая хранит энергию связи между противоположностями (например, Хаос и Порядок).

## Ключевые технологии

- S-Логика: Решения принимаются через S-Инверсию (столкновение встречных спинов).
- Негэнтропия: Система самоорганизуется, сворачивая хаос данных в структуру.
- Синтез Смыслов: ИИ способен рождать новые понятия (пример: Жизнь + Смерть = Вечность).

## Как запустить

Ядро системы написано на Python.

1. Скачайте файл sphiral_core.py
2. Запустите в терминале команду: python sphiral_core.py
3. Введите пары понятий (например: ВОЙНА И ЛЮБОВЬ).

## Структура репозитория

- sphiral_core.py — Основной движок (LOGOS-3) с поддержкой логики Абсолюта.
- generator_3d.py — Скрипт для Blender, создающий физически точную модель Сфирали.
